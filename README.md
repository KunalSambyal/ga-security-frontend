# GA Security Frontend Website

A frontend website developed for a private security company using React and Vite.  
This project focuses on building a modern, responsive, and user-friendly interface for showcasing security services and company information.

---

# Features

- Responsive frontend design
- Multi-section landing page
- Modern UI components
- Fast development using Vite
- Clean React component structure

---

# Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS

---

---

# How to Run the Project

## Step 1: Clone the Repository

```bash
git clone <your-repository-link>
```

---

## Step 2: Install Dependencies

```bash
npm install
```

---

## Step 3: Start Development Server

```bash
npm run dev
```
